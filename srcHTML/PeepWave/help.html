Peep Wave

KEY FUNCTION
[up] switch to previous image
[down] switch to next image
[space] play prompt associated with image
[comma] reveal alt image
[period] peep mode
"[" adjust size of peep window
"]"
[bspace] draw mode
----- ----------------------
[bspace] switch to draw mode
[S]-[bspace] return to normal mode
w return focus to parent frame
