Pick and Place

(actual keys of this instance may vary from the standard key assignment)

KEY FUNCTION
1 to n select piece #n
[S]-[1 to n] place piece #n
p return piece to "pick" location
[S]-p return all pieces
s shuffle pieces
[space] play prompt of active piece
[comma] toggle image of piece
[S]-[comma] toggle all pieces
[period] hold piece for dragging
[right] rotate piece
[left]
----- ----------------------
[bspace] switch to draw mode
[S]-[bspace] return to normal mode
w return focus to parent frame
