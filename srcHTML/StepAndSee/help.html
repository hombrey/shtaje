KEY FUNCTION
a activate new token
x deactivate token
s step token forward
S step token backward
d roll die
n switch to next active token
[space] play prompt of active step
[comma] cycle through views of active step
[up/down] change mat image

[bspace] switch to draw mode
[S]-[bspace] return to normal mode
w return focus to parent frame
